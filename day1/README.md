# Day 1: Hello World in Go

Setting up the development environment and writing the first Go program.

## 📝 Key Concepts Covered

- **Package Declaration**: `package main` as the entry point.
- **Imports**: Importing standardized libraries like `fmt`.
- **Functions**: Using the `main` function as the program's starting point.
- **Output**: Printing text to the terminal using `fmt.Println()`.

## 📂 Files

- [helloworld.go](helloworld.go): The classic "Hello World" example.

## 🚀 How to Run

From the project root:
```bash
go run day1/helloworld.go
```
