# Day 2: Variables and Constants in Go

This directory contains examples of how to declare and use variables and constants in Go.

## 📝 Key Concepts Covered

- **Variable Declaration**: `var name type = value`
- **Short Variable Declaration**: `name := value` (only inside functions)
- **Zero Values**: Understanding default values for different types when no value is provided.
- **Multiple Variable Declaration**: Declaring multiple variables in a single line or block.
- **Constants**: Using `const` to declare unchangeable values.

## 📂 Files

- [variables.go](variables.go): Detailed examples of variable usage and scopes.

## 🚀 How to Run

From the project root:
```bash
go run day2/variables.go
```
