# Day 3: Data Types in Go

This directory explores Go's built-in data types, including integers, floats, booleans, and strings.

## 📝 Key Concepts Covered

- **Numeric Types**: Signed integers (`int`, `int8`, `int16`, `int32`, `int64`), unsigned integers (`uint`, `uint8`, `uint16`, `uint32`, `uint64`), and floating-point numbers (`float32`, `float64`).
- **Booleans**: `bool` type with `true` and `false` values.
- **Strings**: Understanding string declaration and manipulation.
- **Type Conversions**: Converting between different numeric types.

## 📂 Files

- [main.go](main.go): The entry point for this day's practice, calling functions from other files.
- [boolean.go](boolean.go): Practice with boolean logic and operations.
- [integer.go](integer.go): Deep dive into signed and unsigned integer types.
- [float.go](float.go): Working with floating-point numbers and precision.
- [string.go](string.go): Examples of string handling.
- [data-types.go](data-types.go): General overview of common types.

## 🚀 How to Run

From the project root:
```bash
make run FILE=day3-data-types/main.go
```
