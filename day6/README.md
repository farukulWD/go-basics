# Day 6: Functions in Go

This directory explores how to define and use functions in Go, covering everything from basic syntax to advanced return types and recursion.

## 📝 Key Concepts Covered

- **Functional Syntax**: Basic structure of a function: `func Name(params) returnType`.
- **Parameters**: Passing multiple parameters of different types to functions.
- **Return Values**:
  - **Single Return**: Standard way to pass a result back.
  - **Named Return Values**: Defining variable names for the return values in the function signature for clearer code.
  - **Multiple Return Values**: Go's unique ability to return more than one result from a single function call.
- **Recursion**: Implementing functions that call themselves (demonstrated with counting and factorial calculation).

## 📂 Files

- [main.go](main.go): Comprehensive demonstration of function declarations, parameter handling, and recursive logic.

## 🚀 How to Run

From the project root:
```bash
go run day6/main.go
```
