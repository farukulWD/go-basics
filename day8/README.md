# Day 8: Maps in Go

This directory explores **Maps**, Go's built-in data type for storing key-value pairs.

## 📝 Key Concepts Covered

- **Declaration & Initialization**:
  - Using `make(map[keyType]valueType)` to create an empty, ready-to-use map.
  - Using a map literal (`map[string]int{ "key": value }`) for immediate data population.
- **Data Access**: Using keys like a dictionary to look up values.
- **Iteration**: Using the `for range` loop to traverse through keys and their corresponding values.
- **Use Case**: Efficiently mapping identifiers (like user names) to their respective data (like scores).

## 📂 Files

- [main.go](main.go): Basic introduction to map creation and iteration using a scores example.

## 🚀 How to Run

From the project root:
```bash
go run day8/main.go
```
